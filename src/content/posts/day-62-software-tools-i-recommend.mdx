Here is a collection of products and services I use regularly and am grateful for. I recommended these software tools as well cause it makes my work and life easier.

### Development IDEs

[Xcode](https://developer.apple.com/xcode/) :  
For iOS and macOS dev (duh)

[Android Studio](https://developer.android.com/studio):  
For Android development if needed

[Visual Studio Code](https://code.visualstudio.com/):  
The best IDE for web development.

### Development in general

[GitHub](https://github.com/hoyelam):  
Without GitHub I wouldn’t have been a competent engineer with so many open source projects to learn from.

[Paw](https://paw.cloud/):  
The best Mac HTTP client in my eyes. So many possibilities.

[Sourcetree](https://www.sourcetreeapp.com/):  
Makes Git easier to handle with an UI.

[NativeConnect](https://nativeconnect.app/):  
Much faster than App Store Connect when I just want information.

[ColorSlurp](https://colorslurp.com/):  
Easy tool to check the HEX code of an image on your Mac.

### Mac General

[Alfred](https://www.alfredapp.com/) :  
Spotlight on steroids

[iTerm2](https://www.iterm2.com/) :  
Much better than terminal

[Brave Browser](https://brave.com/):  
Fastest browser I’ve used

[Workona](https://workona.com/):  
Such a good tool to organize tabs (You can choose Simple Tab Groups if you use Firefox)

[1Password](https://1password.com/):  
Secure password manager, can also recommend Bitwarden.

[CleanMyMac X](https://macpaw.com/cleanmymac):  
Removes dirt from my Mac and saves a lot of space

### Communication

[Slack](https://slack.com/intl/en-nl/):  
Very good communication tool and lots of bots support

[WhatsApp](https://www.whatsapp.com/?lang=en):  
Sadly the most popular communication tool in The Netherlands.

[Telegram Messenger](https://telegram.org/):  
Private communication

[Discord](https://discord.com/):  
To chat with my gaming friends or find fun communities.

[Spark](https://sparkmailapp.com/):  
A very good email client mostly due to work email accounts

[ProtonMail](https://protonmail.com/):  
The mail I use for all my personal endeavors and sometimes even work

[Twitter](https://twitter.com/):  
For my daily dose of information consumption

### Entertainment

[Spotify](https://www.spotify.com/nl/):  
Music is life

[Pocket Casts](https://www.pocketcasts.com/):  
The best podcast player I’ve used (Overcast was also very good)

[NetNewsWire](https://ranchero.com/netnewswire/):  
RSS feed reader for the Mac

### Productivity

[Adobe Photoshop Lightroom](https://www.adobe.com/products/photoshop-lightroom.html):  
Easiest tool to edit photos with

[PDF Expert](https://pdfexpert.com/):  
My preferred PDF viewer

[Bear App](https://bear.app/):  
Currently the writing app I use, but I can still recommend Notion or Noto

[Things3](https://culturedcode.com/things/):  
The best TODO app ever created

[Ghost](https://ghost.org/):  
Platform where I write and build this site.

### Investing

[DEGIRO](https://www.degiro.nl/):  
Dutch broker for my investments

[Coinbase](https://www.coinbase.com/):  
For my (main) Crypto investments

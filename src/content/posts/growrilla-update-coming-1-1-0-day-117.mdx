In the next update Growrilla and the first one of this year, two new features will be released.

These are the:

-   Notifications Reminder to count a specific event (premium-only)
-   Total Day Count view

![](/images/imported/2021/01/Simulator-Screen-Shot---iPhone-12-mini---2021-01-05-at-20.05.02-8e654f4e02.png)

_Notification Reminder example_

![](/images/imported/2021/01/Eqwbs7wWMAElqkq-17fe40938b.png)

_Total Day Count example_

With the Total Day Count view, you can see how many counts you’ve done for the day. This lets you remind yourself, how you are progressing with your events or even habits.

The notification reminder for specific events is a feature to remind you daily about a specific item. It can also be used for reminding you to perform a certain event.

These features will be available in Growrilla version 1.1.0, the first update of 2021.

* * *

The biggest challenge was the integration of notifications. There are multiple scenarios that I had to take into account. For example, turning on/off notifications from within the app, turning on/off notifications from the iOS settings, and when re-synchronising the added notifications if it was turned on to off to on.

But all doable given enough time 😉.

I honestly think that I am an average developer. If you would have asked me very specific details about software development, I’m sure I would say something wrong.

I can create things from the ground up given time. From web apps to mobile to databases and servers.

But, I still need to research things up.

Sometimes I even forgot how to write a loop and have to search it up.

Why?

I actually don’t know.

Maybe cause I respect other skills rather than knowing how to use a coding practice.

Does this make an average developer?

That is for others to decide.

Assess my skills and portfolio.

I convince with work rather than small talks.

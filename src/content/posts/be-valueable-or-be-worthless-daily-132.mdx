If you can’t provide value, you are worthless.

You have no worth.

The good thing is, is that it should be easy to create value.

There are millions of problems to be solved, and existing solutions to be improved on.

Being a good friend for example is already valuable in my eyes.

Don’t even need to learn hard skills like programming, marketing, sales, or psychology.

If you can be valuable in ANY way, there means you are worth something.

If you aren’t, then you are worthless.

Be valuable, create things and improve on them.

Don’t be worthless.

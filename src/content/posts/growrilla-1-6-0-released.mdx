Today, Growrilla version 1.6.0 is available.

I thought it would be taking a bit longer, but I’ve been testing it myself for several days and the review by Apple was extremely fast.

So I decided to release it immediately.

A good beginning to the week some would say.

* * *

I’ve worked on an additional feature today that improves the user experience of the Count Reports. This feature allows users that enabled notifications to receive a weekly one to notify them that their report is ready.

Hope you all like it!

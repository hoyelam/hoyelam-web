For Xpensemap and CraftCaption there are new updates available. Here are the recent changes.

![](/images/imported/2025/03/android-chrome-192x192-dedfe1fbeb.png)

_Xpensemap Logo_

[**Xpensemap version 2.0.2**](https://apps.apple.com/nl/app/xpensemap-ai-scan-bills/id6471352461?l=en-GB) improvements:

-   Added: Analysis currency list is now sorted alphabetically
-   Added: If category is inside the scanned document, set category automatically
-   Fixed user categories that sometimes remained unsorted

![](/images/imported/2025/03/android-chrome-192x192-1-ba54e73fd2.png)

_CraftCaption Logo_

[**CraftCaption version 1.0.3**](https://craftcaption.com/) improvements:

-   New: Captions and Hashtags suggest captions and hashtags based on your device language by default.
-   New: CraftCaption will request an App Store review if you have copied a caption or hashtag (5 stars would be nice ^^)
-   Added Dutch support 🇳🇱
-   Added Spanish support 🇪🇸
-   Added traditional Chinese support 🇭🇰

Enjoy!
